# App online stellen – Schritt für Schritt

Alles ist schon vorbereitet. Du musst nur 2 Accounts erstellen und ein paar Klicks machen.
Klaus hilft dir dabei, wenn du in einer neuen Sitzung sagst: "Hilf mir die App online zu stellen."

---

## Schritt 1: GitHub-Account erstellen (einmalig)

1. Gehe zu: https://github.com/signup
2. Erstelle einen Account mit deiner E-Mail
3. Fertig!

## Schritt 2: Code auf GitHub hochladen

Das macht Klaus für dich, sobald du eingeloggt bist.
Sag einfach: "Ich hab jetzt einen GitHub-Account, lade den Code hoch."

## Schritt 3: Render-Account erstellen (einmalig)

1. Gehe zu: https://render.com
2. Klicke "Get Started for Free"
3. Wähle "Sign up with GitHub" (dann brauchst du kein extra Passwort)
4. Fertig!

## Schritt 4: App deployen

Das macht Klaus für dich.
Sag einfach: "Ich hab jetzt einen Render-Account, deploy die App."

---

**Geschätzte Zeit: 5 Minuten**
**Kosten: Kostenlos (Render Free Tier)**
